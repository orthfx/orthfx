# Projects

Here are the open-source projects developed by orthfx:

## Orthodox Reader

A minimal, mobile-first Bible reading application featuring continuous scroll, synchronized audio narration, and a distraction-free reading experience. Built with React, TypeScript, and Tailwind CSS.

**Key Features:**
- Progressive Web App with offline support
- Continuous reading from Genesis to Revelation
- Red letter text for Jesus' words
- Audio narration with synchronized highlighting
- Dark/Light mode with customizable fonts
- Reading position tracking

[View on GitHub](https://github.com/ortho-fox/orthodox-reader)

---

## Orthodox NKJV

A Node.js module containing the complete text of the New King James Version (NKJV) Bible with Orthodox canon books, organized in JSON, XML, and Markdown files.

**Key Features:**
- Complete NKJV text in multiple formats
- Orthodox canon books (Tobit, Judith, Wisdom, etc.)
- Conversion scripts between formats
- Easy integration for developers

[View on GitHub](https://github.com/ortho-fox/orthodox-nkjv)

## Pledge

A Next.js subscription payments starter kit for building high-performance SaaS applications.

**Key Features:**
- Secure user management with Supabase
- Stripe Checkout and customer portal integration
- Automatic syncing of pricing plans via webhooks
- PostgreSQL database with Prisma

[View on GitHub](https://github.com/ortho-fox/pledge)

---

## Orthobot

A Discord bot that posts Bible passages when referenced in chat. Simply type a reference like "John 3:16" and the bot will respond with the verse.

**Key Features:**
- Automatic Bible verse lookup
- Support for multiple reference formats
- Range of verses and chapters
- Multiple references in one message

[View on GitHub](https://github.com/ortho-fox/orthobot)

---

## Sites

A multi-tenant platform starter kit for building scalable applications with custom domain support.

**Key Features:**
- Multi-tenancy with unlimited custom domains
- AI-powered Markdown editor
- Image uploads with Vercel Blob
- Dynamic OG cards for each post
- Built with Next.js App Router and Vercel

[View on GitHub](https://github.com/ortho-fox/sites)
