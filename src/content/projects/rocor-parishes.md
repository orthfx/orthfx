# ROCOR Parishes

A comprehensive directory and locator for Russian Orthodox Church Outside of Russia (ROCOR) parishes. Helps Orthodox Christians find nearby parishes and access parish information.

## Overview

This app provides a searchable directory of ROCOR parishes, making it easy for Orthodox Christians to find local parishes, view service times, and access contact information.

## Key Features

### Parish Directory
- Comprehensive listing of ROCOR parishes
- Detailed parish information
- Service times and schedules
- Contact information and addresses

### Search & Discovery
- Search parishes by name or location
- Location-based parish finding
- Filter and sort capabilities
- Mobile-friendly interface

### User Experience
- Clean, intuitive design
- Fast search and filtering
- Responsive layout for all devices
- Accessible to all users

## Tech Stack

- Vite for build tooling
- React with TypeScript
- Tailwind CSS for styling
- Modern web standards
